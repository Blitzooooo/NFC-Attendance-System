## 8. NTP ЪБМфЦЋВю (OFS) ЛњжЦ

УПЬѕЫЂПЈМЧТМаЏДј `timeOffset` зжЖЮ (REC а­вщ `OFS=%d`)ЃК

| OFS жЕ | КЌвх | ЩЯЮЛЛњДІРэ |
|:---:|------|------|
| `-1` (0xFFFF) | RTC ЩаЮДО­ NTP аЃзМ, ЪБМфВЛПЩаХ | **ЖЊЦњ** |
| `0` | RTC вбаЃзМ, МЧТМЪБМф = БъзМЪБМф | жБНгЪЙгУ |

1. NTP ГЩЙІКѓ Ёњ `Record_FixAllTimeOffsets(delta)` БщРњ Flash ШЋВПМЧТМ

3. ВСГ§МЧТМЩШЧј Ёњ ЛиаД Ёњ OFS ЙщСу (аТМЧТМЮоашаое§)
